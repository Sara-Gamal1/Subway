<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone https://github.com/Sara-Gamal1/graphics-project
```

<p>2. Build using CMake</p>

<p>3. Run game executable</p>

```
./bin/GAME_APPLICATION.exe
```
